# INTRODUCING BAMAZON TO LIRI! #

## Your favorite one-stop node.js shop is now available on Liri. ##
Liri is a backend nodeJS Siri clone.

* NodeJS app
* NPM Packages
* Multiple API
* MySQL Database

### Features include ###
1. GAMES
   * ClozeCard
   * BasicCard
1. BAMAZON SHOPPING
   * Customer view
   * Manager view
   * Supervisor view
1. Twitter API
   * See your tweets!
1. Spotify API
1. OMDB API
   * Search by year
